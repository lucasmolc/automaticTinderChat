## Tarefa
Analise os dados de conversas e gere insights acionáveis para melhorar a taxa de sucesso.

## Dados para Análise
{analytics_data}

## Métricas Disponíveis
- Total de matches
- Taxa de resposta às primeiras mensagens
- Tempo médio até primeira resposta
- Taxa de conversão para WhatsApp
- Taxa de conversão para encontros
- Tipos de mensagens mais eficazes
- Horários com maior taxa de resposta

## Análise Necessária

### 1. Performance Geral
- Como está a taxa de resposta?
- Qual o padrão de engajamento?

### 2. Análise de Mensagens
- Quais tipos de primeira mensagem funcionam melhor?
- Quais abordagens têm mais sucesso?

### 3. Gargalos Identificados
- Onde estão as maiores perdas?
- O que pode ser melhorado?

### 4. Recomendações
- Sugestões para o perfil
- Sugestões para a bio
- Sugestões de abordagem
- Melhores horários para usar o app
