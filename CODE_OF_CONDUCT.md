# Código de Conduta

Este projeto adota o [Contributor Covenant v2.1](https://www.contributor-covenant.org/pt-br/version/2/1/code_of_conduct/).

## Resumo

- Seja respeitoso e acolhedor com todos os participantes
- Críticas são sobre código e ideias, nunca sobre pessoas
- Não serão tolerados assédio, ataques pessoais, linguagem discriminatória ou divulgação de informações privadas de terceiros
- Em especial neste projeto: **nunca compartilhe dados reais de pessoas** (matches, conversas, fotos) em issues, PRs ou discussões

## Aplicação

Violações podem ser reportadas de forma privada aos mantenedores via GitHub. Os mantenedores podem remover conteúdo e banir participantes que violem este código.
